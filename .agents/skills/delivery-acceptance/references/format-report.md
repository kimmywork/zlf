# Format Fit: Reports & Analysis

Use when reviewing reports, analysis docs, or written deliverables. Check against these criteria in addition to Spec Fit.

## Factual Accuracy

- [ ] Every factual claim cites a source (document, URL, data, expert statement).
- [ ] Sources are graded: official docs / peer-reviewed / preprint / gray lit / unverified.
- [ ] Unverifiable claims are marked as "assertion" or "from-agent-knowledge."
- [ ] Conflicting sources are noted and explained.

## Structure & Clarity

- [ ] Executive summary captures key findings in 2-3 sentences.
- [ ] Sections are logically ordered and clearly headed.
- [ ] Terminology is consistent throughout.
- [ ] Length is appropriate for the audience (no padding).

## Traceability

- [ ] Data, quotes, and references can be traced to their origin.
- [ ] Methodology is described (how the information was gathered).
- [ ] Limitations and known gaps are explicitly stated.
