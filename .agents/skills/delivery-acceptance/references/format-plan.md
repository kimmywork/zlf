# Format Fit: Plans & Proposals

Use when reviewing plans, proposals, or solution designs. Check against these criteria in addition to Spec Fit.

## Feasibility

- [ ] Every task or step is actionable with available resources.
- [ ] Dependencies are identified and resolved or documented.
- [ ] Timeline estimates are realistic and bounded.
- [ ] Rollback or contingency is defined for critical paths.

## Completeness

- [ ] Every requirement from the source document is addressed.
- [ ] Non-goals are respected (no unapproved scope addition).
- [ ] Risks are identified with mitigation strategies.
- [ ] Verification method is defined for each deliverable.

## Consistency

- [ ] No contradictory steps or assumptions within the plan.
- [ ] Terminology and naming are consistent with existing conventions.
- [ ] Dependencies between tasks are logically ordered.
